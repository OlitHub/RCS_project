rcs_project.rcs_fun.get_color(img, coord)¶

Returns the color of the pixel at the coordinates (x,y) of the image

Args:

x (float): Coordinate X of the pixel y (float): Coordinate Y of the pixel img (PIL image): Image to be used as a size reference

Returns:

color (tuple): Tuple of the RGB values of the pixel

rcs_project.rcs_fun.get_color2(img, center, hex_size)¶

Returns the average color of the hexagon whose center is center and size is hex_size

Args:

img (PIL image): Image center (tuple(x, y)): Coordinates of the center of the hexagon hex_size (float): Size of the hexagon

Returns:

pixel (tuple): Tuple of the RGB values of the average color of the hexagon
